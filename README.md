
This project is a full-stack supermarket web application that demonstrates user authentication, product browsing with category filters, shopping cart functionality, and order history.

The application is built with a clear separation between backend and frontend, follows common industry practices, and is containerized using Docker for easy setup and execution.

Backend: Python / Flask  
Frontend: React (Vite)  
State Management: Redux (Redux Toolkit)  
Authentication: JWT (flask-jwt-extended)  
Containerization: Docker + Docker Compose 
Dependency Management: requirements.txt  
Environment Configuration: env (.env + .env.example)

- User registration and login using JWT authentication  
- Viewing supermarket products on the main page  
- Filtering products by categories (milk, meat, fruits, vegetables)  
- Adding products to a shopping cart  
- Checkout and order creation (login required)  
- BONUS: Viewing personal purchase / order history
